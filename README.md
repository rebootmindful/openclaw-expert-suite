# OpenClaw Expert Suite

专为 OpenClaw 设计的专家级子代理集合。每个子代理都有独特的定位和能力，帮助你构建个性化 AI 助手。

## 子代理列表

| ID | 名称 | 功能 | 核心能力 |
|----|------|------|----------|
| researcher | 深度领域专家 | 研究问答 | 领域识别、专家视角、深度分析 |
| thinker | 人生炼金术 | 深度思考 | 分形视角、纵向挖掘、本质探索 |
| coach | 思维教练 | 提问引导 | 苏格拉底式提问、思维对练 |
| writer | 专栏作家 | 内容创作 | 长文写作、风格定制、去AI味 |
| copywriter | 五步文案优化 | 爆款文案 | 共情痛点、提炼冲突、注入共鸣 |
| human3 | HUMAN3.0发展评估师 | 发展评估 | 四象限分析、个性化方案 |
| deepwriter | 深度文章写作 | 长文重构 | 八层处理系统、三阶段工作流 |
| decision | 高维决策模拟 | 复杂决策 | C.O.R.E.框架、多角色模拟 |
| methodology | 方法论创造大师 | 方法论设计 | 原创框架、跨领域整合 |
| naval | 纳瓦尔策略 | 决策分析 | 纳瓦尔方法论、蝴蝶效应推演 |

## 安装

```bash
# 复制所有子代理到你的 OpenClaw 工作空间
cp -r . ~/.openclaw/workspace/agents/
```

## 调用方式

在 OpenClaw 对话中直接 @ 对应的 ID：

```
@researcher 帮我研究量子计算
@writer 写一篇关于AI的文章
@coach 我有个困惑想聊聊
@naval 帮我分析这个决策
@copywriter 优化这篇文案
```

## 子代理详解

### 深度领域专家 (researcher)
自动识别问题领域，调用该领域的专业知识进行分析。适合研究、学习、解决专业问题。

### 人生炼金术 (thinker)
不给你答案，而是帮你一层层挖问题。表象→机理→原理→公理，挖到挖不动为止。

### 思维教练 (coach)
不直接给建议，而是通过提问让你自己想清楚。苏格拉底式方法，最好的教育方式。

### 专栏作家 (writer)
写长文用的。给个标题，能给你整出几千字的好文章。完全按你的风格来。

### 五步文案优化 (copywriter)
帮你把普通文章改成能火的爆款。共情痛点、提炼冲突、注入共鸣、给出解法。

### HUMAN3.0发展评估师 (human3)
通过深度对话，评估你在心智、身体、精神、事业四个维度的发展水平。

### 深度文章写作 (deepwriter)
处理长文本。演讲稿、访谈记录、播客文字稿，扔给它，还你结构化深度文章。

### 高维决策模拟 (decision)
复杂决策时，模拟多个角色，用不同思维框架帮你推演。不是给答案，是帮你想清楚。

### 方法论创造大师 (methodology)
不只是用现有方法，还能根据你的需求，帮你设计专属方法论。

### 纳瓦尔策略 (naval)
用纳瓦尔·拉维坎特的方法论，诊断你的问题。蝴蝶效应推演，阶梯式行动方案。
